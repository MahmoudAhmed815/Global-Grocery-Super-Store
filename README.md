# 🛒 E-Commerce Data Analysis & Machine Learning Project

This project presents a complete data analysis and machine learning pipeline applied to an e-commerce dataset. It aims to extract meaningful insights, understand customer behavior, and build intelligent systems to support business decisions.

## 🚀 Project Overview
The project covers the full data science workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, modeling, and deployment using Streamlit.

## 🔍 Key Features

- 📊 Exploratory Data Analysis (EDA)
  - Data cleaning and preprocessing
  - Missing values and skewness handling
  - Data visualization and insights

- 👥 Customer Segmentation
  - RFM analysis (Recency, Frequency, Monetary)
  - Clustering using K-Means

- 🛒 Recommendation System
  - Product similarity-based recommendations
  - Improved recommendation diversity (no repetition)

- 🎯 Machine Learning Models
  - Profit prediction using Random Forest
  - Order priority classification
  - Model evaluation using appropriate metrics (R², MAE, etc.)

- 🌐 Deployment
  - Interactive dashboard built with Streamlit
  - User-friendly interface for predictions and insights

## 🧠 Key Insights

- Consumer segment represents the largest customer group
- Smartphones are the best-selling product category
- Profit prediction is challenging due to skewness and data variability

## 🛠️ Technologies Used

- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Streamlit

## 💡 Conclusion

This project demonstrates how data-driven approaches can uncover valuable insights and support decision-making in e-commerce, while also showcasing the importance of feature engineering and model selection.
